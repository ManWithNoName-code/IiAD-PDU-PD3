# Tutorial
1. Download/clone all files from this repository.
2. Open folder with in terminal.
3. Type:
```console
./requirements.sh
```
4. If it doesn't work try:
```console
sudo chmod +x requirements.sh
```
   and then again:
```console
./requirements.sh
```
5. Now you can open and use our R program with full functionality!
